CodeScale is an experiment to determine the cognitive weights of basic control structures in imperative programming languages based on user perception and response times.

The experiment comprises the following stages:

  • Java Refresher: this helps participants get up to speed with basic java control structures. Participants are presented with code snippets in Java and their corresponding output.
  • Main Activity: at this stage, participants will be presented with Java code snippets and will be required to provide the expected output via a text input field.
  • Survey: A brief survey would be presented to the participant.

In order to ensure the accuracy of the experiment, please ensure that you do not get distracted or interrupted throughout the course of your participation.
