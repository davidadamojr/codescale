CodeScale is an experiment to determine the cognitive weights of basic control structures in imperative programming languages based on user perception and response times.

The experiment comprises the following stages:

  • Practice Sessions: this helps participants get up to speed with basic java control structures. Participants are presented with code snippets in Java and their corresponding output.
  • Main Activity: at this stage, participants will be presented with Java code snippets and will be required to provide the expected output via a text input field.
  • Survey: A brief survey will be presented to the participant.

In order to ensure the accuracy of the experiment, please ensure that you do not get distracted or interrupted while working on a code snippet. You can take breaks in between activities/code snippets

You may use a pen and paper to help you work through the code snippets.
